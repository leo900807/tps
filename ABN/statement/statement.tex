\section*{\bd{第二題：A+B=N(ABN)}}
\subsection*{\bd{問題敘述}}

相信有在打競賽的各位，都聽過 A+B Problem 吧，就是給你 $A$ 跟 $B$，請你輸出 $A + B$ 的值，我們現在把這題做一點變化，給你 $A + B$ 的值 $N$，請你構造出 $M$ 組不同的 $A$ 和 $B$，使得這 $M$ 組的 $A+B=N$ 。

不同的定義為 $A$ 或 $B$ 任一個不同，即視為不同解，也就是說 $3 + 2 \ne 2 + 3$。

\bd{請注意：你輸出的 $A$ 和 $B$ 必須為整數且符合 $0 \le A, B \le 10^9$。} 

\subsection*{\bd{輸入格式}}
輸入只有一行，包含兩個整數 $N$ 和 $M(M \le \min(100, N))$，分別如題目所述。

\subsection*{\bd{輸出格式}}
請輸出 $M$ 行，每行包含兩個正整數 $A$ 和 $B$，兩個正整數用一個空格隔開，$A$ 和 $B$ 分別如題目所述，若輸出超過 $M$ 行、輸出不該輸出的字元或是輸出的數字範圍不符合題目要求，該筆測試資料將被視為 \bd{Wrong Answer}。

\subsection*{\bd{測資限制}}
\begin{itemize}
    \item $2\le N \le 10^9$。
    \item $1\le M \le \min(100, N)$。
\end{itemize}

\subsection*{\bd{輸入範例 1}}
{
\setlength\parindent{0pt}
\td{
5 3
}
}
\subsection*{\bd{輸出範例 1}}
{
\setlength\parindent{0pt}
\td{
1 4\\
2 3\\
3 2
}
}
\subsection*{\bd{輸入範例 2}}
{
\setlength\parindent{0pt}
\td{
16 8
}
}
\subsection*{\bd{輸出範例 2}}
{
\setlength\parindent{0pt}
\td{
1 15\\
2 14\\
3 13\\
4 12\\
5 11\\
6 10\\
7 9\\
8 8
}
}
