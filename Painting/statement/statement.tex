\section*{\bd{第一題：繪圖(Painting)}}
\subsection*{\bd{問題敘述}}
你喜歡畫畫嗎？現在給你一張 $N\times M$ 的空白繪板，請你盡可能的在上面畫出你喜歡的圖案吧！
\subsection*{\bd{輸入格式}}
首行輸入兩個整數 $N,M$ 以空格隔開。
\subsection*{\bd{輸出格式}}
輸出一個 $N \times M$ 的 $01$ 表格，$0$ 的部份代表留白， $1$ 的部份代表你在表格上的筆跡。

請注意，我們不允許偷懶，所以\bd{請勿繳交一份全白或全黑的繪板}，此外我們也厭惡抄襲，所以你畫的圖案也\bd{請勿與任何一筆輸出範例相同}。
\subsection*{\bd{測資限制}}
\begin{itemize}
    \item $1\le N\le 100$。
    \item $1\le M\le 100$。
    \item $2\le N\times M\le 10000$。
\end{itemize}
\subsection*{\bd{輸入範例 1}}
{
\setlength\parindent{0pt}
\td{
2 2
}
}
\subsection*{\bd{輸出範例 1}}
{
\setlength\parindent{0pt}
\td{
01\\
10
}
}
\subsection*{\bd{輸入範例 2}}
{
\setlength\parindent{0pt}
\td{
5 19
}
}
\subsection*{\bd{輸出範例 2}}
{
\setlength\parindent{0pt}
\td{
1111011110111101001\\
1001010000100001001\\
1111010000111101111\\
1000010000000101001\\
1000011110111101001
}
}
